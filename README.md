# 🌊 RiverFlow - Full Stack Q&A Platform

A modern full-stack Question & Answer platform inspired by Stack Overflow, where users can register, ask questions, answer community posts, vote, search content, and manage their profiles.

## 🚀 Live Demo

🌐 Live Website: https://26-full-stack-qna-system.vercel.app

📂 GitHub Repository: https://github.com/Vinayak-123-jpj/riverflow-qa-platform

---

## ✨ Features

- 🔐 Secure Authentication using Appwrite
- 👤 User Registration & Login
- 📝 Ask, Edit and Delete Questions
- 💬 Answer Questions
- 👍 Upvote & Downvote System
- 🔍 Search Questions
- 🏷️ Tag Support
- 🖼️ Image Upload
- 👤 User Profile

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Vinayak-123-jpj/riverflow-qa-platform.git
```

Move into the project

```bash
cd riverflow-qa-platform
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add:

```env
NEXT_PUBLIC_APPWRITE_HOST_URL=https://cloud.appwrite.io/v1

NEXT_PUBLIC_APPWRITE_PROJECT_ID=YOUR_PROJECT_ID

APPWRITE_API_KEY=YOUR_API_KEY
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
